# 🌿 Floravision Frontend

A pixel-perfect and fully responsive frontend implementation of the provided Figma design for a modern plant landing page.

Built using Tailwind CSS with a premium glassmorphism-inspired UI and responsive layouts.

---

# 📌 Project Overview

Floravision Frontend is a modern plant e-commerce landing page developed by replicating the provided Figma design as accurately as possible.

The project focuses on:

- Pixel-perfect UI implementation
- Responsive design across all devices
- Reusable frontend sections
- Smooth hover animations
- Modern glassmorphism effects
- Clean and production-ready code

---

# 🛠️ Tech Stack

- HTML5
- Tailwind CSS(using CLI)
